## Lições para o Claude futuro

1. **Imagens directas Discord são anexos, não mãos.** Documentado em `CLAUDE.md` secção "Imagens de contexto Discord". Não tratar `entry_type='image'` como hand source. Ler antes de tocar em qualquer pipeline Discord.

2. **Janela de match imagem ↔ mão: ±90 segundos.** Não 5min, não 10min. Janelas mais largas trazem ruído inaceitável (cross-talk entre mãos consecutivas).

3. **Antes de propor backfill, verificar a 100% que o fix do bug em produção bloqueia recorrência.** Caso 25 mãos: confirmado via deploy + queries antes do `--execute`. Sem isto, o backfill é tapar o sol com a peneira — novo bug acumula no mesmo padrão.

4. **Investigação por SQL contra prod via `railway run --service Postgres python` é o caminho confiável.** CLI `psql` não está instalado localmente. `psycopg2` directo via `railway run python` falha com `UnicodeDecodeError` por causa dos PG* env vars no Windows. Solução: strip PG* + usar `DATABASE_PUBLIC_URL` + workaround `dsn.encode("cp1252").decode("utf-8")`. Padrão já estabelecido em `backfill_villains.py` e `backfill_revert_placeholder_match.py`.

5. **`Hands.jsx` tem gates de renderização condicionais que assumem matched>0** — verificar antes de adicionar features que devem aparecer quando matched=0. Caso real: gate `tagGroupsData.groups.length > 0` em `:1858` (agora `:1857`) escondia secção nova quando matched=0; descoberto via logs temporários no browser.

6. **Quando o Rui pergunta "já te disse isto", ouvir.** Documentar permanentemente em `CLAUDE.md` (e para conceitos derivados, em `MAPA_ACOPLAMENTO.md`), não apenas confirmar e continuar. Caso real: regra "imagens Discord são anexos" foi mencionada em sessões anteriores, esquecida; só ficou permanente após pedido explícito de documentação a 26-Abr.

7. **Decisões em linguagem técnica perdem o Rui rapidamente.** Apresentar opções em linguagem normal primeiro (PT-PT, sem jargão), com a tradução técnica depois. Padrão validado quando opções (a)/(b)/(c) foram apresentadas em prosa antes do código — Rui escolhe consistentemente quando consegue ler em pt-pt corrente. Padrão a manter.
