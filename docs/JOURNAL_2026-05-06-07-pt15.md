# Journal pt15 — 2026-05-06 e 2026-05-07

## Contexto

Sessão dedicada a iteração visual exclusiva do frontend — UI/UX. Sem mudanças de backend, parsers, schema ou dados. O foco foi a apresentação dos cabeçalhos de torneio (TournamentHeader), o popup do replayer (ReplayerPage) e a uniformização das cartas de poker em todos os componentes.

A sessão durou 2 dias civis (06 e 07 Maio) e produziu ~25 commits no `main` ramo, todos cosméticos / UX.

## Trabalho realizado

### Painel torneio (TournamentHeader + Hands.jsx)

**Refactor flex layout completo (commit `33d9791`)**
- Wordmark/halo deixaram de ser `position: absolute`, passaram a flex items in-flow.
- `customTitle` expande com `flex 1 1 auto`.
- Halo+wordmark agrupados em "wordmark area" (480×120, position relative, halo+stars+wordmark `inset:0` dentro).
- Stats também passaram a flex item (180px, justify-end).
- Buffers entre items via `gap: 12px` do flex parent — eliminados todos os `marginLeft/marginRight` manuais.

**GG logo horizontal**
- `gg_horizontal.png` (320×160) substitui `gg1.png` (vertical 1:1).
- Fundo removido com PIL no asset.

**Estudo bareMode — design completamente novo**
- Sem halo + wordmark grande visível.
- Watermark da sala (texto puro, opacity 0.10) com gradient horizontal "emergente da direita":
  - WPN/Ya: verde (`rgba(107,142,35,…)`).
  - Winamax: vermelho (`rgba(220,40,50,…)`).
  - PokerStars: branco (`rgba(255,255,255,…)`).
  - GGPoker: azul-petróleo (`rgba(45,110,145,…)`).
- Watermark renderizada DENTRO da 7ª cell do grid (centrada via prop `centerInCell`).
- Watermark do Ya: split em 2 spans (Ya verde + Poker branco).
- bareMode aplica-se SÓ a Estudo. HM3 e Tournaments mantêm modo normal (halo + wordmark grande).
- Iterações: localized gradient → vertical gradient → horizontal emergente (commits `5364908`, `6073c11`, `518af56`, `8529cb5`, `4157e1e`, `d5b3dae`).

**Fix corte vertical da watermark**
- Wrapper interno do TournamentHeader tinha `overflow:hidden + textOverflow:ellipsis + whiteSpace:nowrap` em todos os modos. Quando `customTitle != null`, agora omite as 3 propriedades. Default mode (HM3, Tournaments) preserva ellipsis.

**fontSizes customTitle aumentados**
- Nome 14→18, BI 13→15, Data 12→14, Mãos 11→14. TM e SI mantêm 11.
- Card `minHeight` bareMode 90→110.

**`cleanTournamentName` — fixes WPN e GG**
- WPN: preserva `$X GTD` prefix; remove só `Tournament` trailing. Ex: `"$75,000 GTD Tournament"` → `"$75,000 GTD"`.
- GG: regex alargada para BI com vírgula separadora (`$1,050` / `$1,050,000`). Boundary aceita vírgula trailing. Cleanup de leftovers (`,\s*,`, `\s*,$`, etc). Ex: `"GGMasters High Rollers $1,050, 750K GTD"` → `"GGMasters High Rollers, 750K GTD"`.

G. **Cores per-player por tipo de aposta**: `getPlayerAggressionLevels()` infere nível na street.
   - Pré: 1→`#F5C16C` (open), 2→`#F97316` (3-bet), 3→`#DC2626` (4-bet), 4+→`#A855F7` (5-bet+).
   - Pós: 1→`#FBBF24` (cbet), 2→`#F97316` (raise), 3+→`#DC2626` (re-raise).

- Pot 28→30. Board 40→44.

**Fix chip stacks dentro do felt + hero não colide (commit `7fcfe6f`)**

- `CHIP_RATIO_HERO`=0.45 (Hero ratio especial — cards xl extendem-se acima E abaixo do `player_y` por causa do `translate(-50%,-50%)`; 0.45 mantém chip pile acima das cards).

**Fix badges D/SB/BB (último commit `56054d3`)**
